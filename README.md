## Development

```bash

$ pre-commit install --hook-type commit-msg

# install deps
$ npm install

# start local development server
$ npm run dev
```

## Unit testing

You can run unit testing by running

```bash
# while watching changes
$ npm test

# without watching changes
$ npm test run
```
