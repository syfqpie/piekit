# FormTextArea

`FormTextArea` is a customizable textarea component that standardizes `textarea` behavior and styling while ensuring a consistent user experience.

## Import

```tsx
import { FormTextArea } from 'piekit'
```

## Props

### label

Default value: `undefined`\
Type: `string`\
Description: The `label` prop sets a text label above the teaxtare field.

<Canvas of={FormTextAreaStories.LabelOnly} />

### placeholder

Default value: `undefined`\
Type: `string`\
Description: The `placeholder` prop sets a temporary hint inside the textarea field, displayed when it is empty.

<Canvas of={FormTextAreaStories.PlaceholderOnly} />

### disabled

Default value: `false`\
Type: `boolean`\
Description: If `disabled` prop is set to `true`, the textarea will be disabled.

<Canvas of={FormTextAreaStories.Disabled} />

### hasError

Default value: `false`\
Type: `boolean`\
Description: If `hasError` prop is set to `true`, the textarea appearance will change to `error` variant.

<Canvas of={FormTextAreaStories.ErrorStateWithoutMessage} />

### errorMessages

Default value: `[]`\
Type: `string[]`\
Description: If `errorMessages` prop is being set, error messages will be shown. `hasError` have to be set to `true`.

<Canvas of={FormTextAreaStories.ErrorStateWithMessage} />

### groupClassName

Default value: `undefined`\
Type: `string`\
Description: The `groupClassName` prop allows you to apply additional styling to the container that wraps the textarea field and label.

<Canvas of={FormTextAreaStories.WithGroupClassName} />

## Test automation tag

To assist with automated testing, add a `testId` attribute to the `FormTextArea`.

<Canvas of={FormTextAreaStories.WithTA} />

Will be rendered as:

```html
<div data-testid="name-form-ta-form">
    <label data-testid="name-form-ta-label">Name</label>
    <textarea placeholder="Enter name" data-testid="name-form-ta-textarea"></textarea>
    <p data-testid="name-form-ta-error-msg-0">Field required</p>
</div>
```
