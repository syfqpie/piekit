# Card

The `Card` component is a flexible container used to group related content in a clean layout.

## Import

```tsx
import { Card } from 'piekit'
```

## Basic usage

<Canvas of={CardStories.Default} />

## Test automation tag

To assist with automated testing, add a `testId` attribute to the `Card`.

<Canvas of={CardStories.WithTA} />

Will be rendered as:

```html
<div data-testid="card-ta">
    content
</div>
```
