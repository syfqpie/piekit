# FormSelect

`FormSelect` is a customizable select component that standardizes `select` behavior and styling while ensuring a consistent user experience.

## Import

```tsx
import { FormSelect } from 'piekit'
```

## Basic usage

```tsx
<FormSelect
    label='Fruit'
    placeholder='Select fruit'
    value={value}
    onChange={(e) => {setValue(e.value)}} />
```

## Props

### label

Default value: `undefined`\
Type: `string`\
Description: The `label` prop sets a text label above the select field.

<Canvas of={FormSelectStories.LabelOnly} />

### placeholder

Default value: `undefined`\
Type: `string`\
Description: The `placeholder` prop sets a temporary hint inside the select field, displayed when it is empty.

<Canvas of={FormSelectStories.PlaceholderOnly} />

### disabled

Default value: `false`\
Type: `boolean`\
Description: If `disabled` prop is set to `true`, the select will be disabled.

<Canvas of={FormSelectStories.Disabled} />

### hasError

Default value: `false`\
Type: `boolean`\
Description: If `hasError` prop is set to `true`, the select appearance will change to `error` variant.

<Canvas of={FormSelectStories.ErrorStateWithoutMessage} />

### errorMessages

Default value: `[]`\
Type: `string[]`\
Description: If `errorMessages` prop is being set, error messages will be shown. `hasError` have to be set to `true`.

<Canvas of={FormSelectStories.ErrorStateWithMessage} />

### groupClassName

Default value: `undefined`\
Type: `string`\
Description: The `groupClassName` prop allows you to apply additional styling to the container that wraps the select field and label.

<Canvas of={FormSelectStories.WithGroupClassName} />

## Test automation tag

To assist with automated testing, add a `testId` attribute to the `FormSelect`.

<Canvas of={FormSelectStories.WithTA} />

Will be rendered as:

```html
<div data-testid="select-form-ta-form">
    <label data-testid="select-form-ta-label">Fruit</label>
    <button data-testid="select-form-ta-toggler">Select fruit</button>
    <div>
        <ul data-testid="select-form-ta-options-container">
            <li>
                <button data-testid="select-form-ta-option-0">...</button>
            </li>
        </ul>
    </div>
    <p data-testid="select-form-ta-error-msg-0">Field required</p>
</div>
```
